Javier Vargas

Descripción del proyecto

DUAD To-Do es una Single Page Application que permite a múltiples usuarios gestionar sus listas de tareas de forma independiente. Cada usuario puede crear, editar, completar y eliminar tareas. La sesión persiste entre recargas gracias a localStorage, y las tareas se almacenan en tiempo real usando la API pública de restful-api.dev.

Stack técnico

Vanilla JS Lógica principal
HTML5 Estructura
CSS3 Estilos
Axios HTTP requests
localStorage Sesión & usuarios
restful-api.dev API de tareas
Google Fonts Syne + DM Sans

Funcionalidades implementadas

Sistema de autenticación con registro e inicio de sesión simulado en el navegador. CRUD completo de tareas con integración a API externa. Filtrado por estado (todas, pendientes, completadas). Edición inline con doble click. Protección de rutas. Diseño responsivo con tema oscuro y accesible.
